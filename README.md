# tray_number
Show CPU usage or temperature as a number in Windows System Tray.

It is based on Python 3 and uses library DLL OpenHardwareMonitor.
In first run will generate a file 'sensors.txt' where you can discover the name of your devices read by the library.

This script shows the CPU temperature close to the clock in Windows.
It is useful to monitor CPU usage and performance during the day, so you can find bottlenecks in your PC.

Future versions will have better appearance and no open window.
